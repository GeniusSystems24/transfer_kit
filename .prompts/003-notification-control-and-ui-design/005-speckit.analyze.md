/speckit.analyze

Analyze notification control and design specification, plan, and tasks.

Check:

1. Is notification scope clear?
2. Is default behavior clear?
3. Is permission behavior clear?
4. Is notification package coupling isolated?
5. Are fake adapter tests included?
6. Is progress notification throttling defined?
7. Is duplicate prevention defined?
8. Is batch notification behavior defined?
9. Are notification actions included or explicitly deferred?
10. Are platform limitations documented?
11. Does this task avoid Firebase removal?
12. Does this task avoid cache policy changes?

Return:
- issue
- risk
- correction
